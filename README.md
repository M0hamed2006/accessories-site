## ⚙️ التخصيص والتحديث

### 1️⃣ تغيير رقم الواتساب:

في **script.js**، ابحث عن:
```javascript
const CONFIG = {
    WHATSAPP_NUMBER: "201118618196",
    ...
};
```

غيره إلى رقمك:
```javascript
const CONFIG = {
    WHATSAPP_NUMBER: "20XXXXXXXXXX",  // ضع رقمك هنا
    ...
};
```

### 2️⃣ تعديل المنتجات:

في **index.html**، ابحث عن:
```html
<div class="product-card premium-card">
    <h4>اسم المنتج</h4>
    <p class="product-description">الوصف</p>
    <button onclick="order('CODE', 'NAME', 250)">
```

غيّر:
- `CODE`: كود المنتج الفريد
- `NAME`: اسم المنتج
- `250`: السعر بالجنيه المصري

### 3️⃣ تغيير الألوان:

في **style.css**، ابحث عن `--primary`:
```css
:root {
    --primary: #c1a072;           /* اللون الأساسي */
    --accent-gold: #ffd700;       /* ذهبي */
    --accent-rose: #f4a5d5;       /* وردي */
    --accent-purple: #9b59b6;     /* بنفسجي */
    /* ... الألوان الأخرى */
}
```

### 4️⃣ إضافة صور حقيقية:

استبدل الـ placeholders في **style.css**:
```css
.gradient-gold {
    background: url('path/to/your/image.jpg');
    background-size: cover;
}
```

### 5️⃣ تعديل النصوص والمعلومات:

في **index.html** غيّر:
- الوصف في hero section
- قصتك في قسم "عننا"
- روابط السوشيال ميديا
- معلومات التواصل
